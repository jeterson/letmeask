# Projeto NLW missão ReactJS
## Descrição
O projeto consiste em desenvolver uma ferramenta que apoie streamers a responder 
perguntas em suas lives

## O que usei ?
* ReactJS
  * Hooks
  * Contexts
  * Routes
* TypeScript
* SCSS
* Firebase
  * Realtime Database
  * Hosting
  * Authentication
